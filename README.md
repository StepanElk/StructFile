# **Struct File Helper**  

## **Сборка**  
### **Требования**  
- Компилятор C++17 (GCC)  
- CMake 3.10+  

### **Инструкция**  
1. Создать и собрать проект:  
   ```bash
   mkdir build && cd build
   cmake ../ && cmake --build ./
   ```  

2. Запустить программу:  
   ```bash
   ./struct_file_helper
   ```
